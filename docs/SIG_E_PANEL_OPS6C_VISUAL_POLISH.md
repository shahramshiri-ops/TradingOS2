# SIG-E-PANEL-OPS6C — Visual Polish

This patch polishes OPS6B based on the screenshot review.

Fixes:
- Adds update-age indicator to the right side of the tab menu:
  `Updated 2m ago` style.
- Adds exact UTC refresh time beside it on desktop.
- Forces a dark-first color system because the screenshot showed low contrast in light mode.
- Improves card contrast, hierarchy, spacing, active tabs and button states.
- Keeps OPS6B tabs and data behavior unchanged.

No detector, portfolio, memory, signal, or execution logic is changed.
