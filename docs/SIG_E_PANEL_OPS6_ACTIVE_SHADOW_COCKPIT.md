# SIG-E-PANEL-OPS6 — Active Shadow Research Cockpit

Professional redesign of the Brain4/SIG-E panel for the current project direction.

The panel becomes a simple cockpit for live shadow research:
- System state
- Active shadow events, if any
- Lane cards for the 4-lane portfolio
- Coverage status
- Refresh / near-miss / shadow-event counters
- Clear authority boundary

No detector, portfolio, memory, signal, or execution logic is changed.

Boundary:
Display only. Shadow research only. Not a signal. No trade proposal. No entry/stop/target. No risk sizing. No broker/execution. No auto execution.
