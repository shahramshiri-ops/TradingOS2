# Active Watch Reference Levels

این بخش به Active Watch کمک می‌کند که فقط نگوید «فعال است»، بلکه نشان دهد observation با چه سطح‌هایی خوانده می‌شود:

- محدودهٔ نامعتبر شدن مشاهده
- مرجع ضعیف‌شدن
- مرجع پیشروی
- محدودهٔ امتداد تقریبی observation

این سطح‌ها trade target یا stop نیستند. فقط برای فهمیدن وضعیت observation هستند.
